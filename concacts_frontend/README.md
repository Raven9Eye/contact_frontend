# 通讯录前端项目

这是通讯录应用的前端部分，使用HTML、CSS和JavaScript实现。

## 功能特性

- 添加联系人
- 修改联系人
- 删除联系人
- 查看联系人列表

## 目录结构

```
|- src/
|  |- index.html
|  |- styles.css
|  |- app.js
|  |- api.js
|- README.md
|- codestyle.md
```

## 快速开始

1. 克隆仓库
2. 在浏览器中打开 `src/index.html`

## 技术栈

- HTML5
- CSS3
- JavaScript
- Fetch API
